Interest Calculation Form







Steps to Calculate the Interest

  • Fill from Top to Bottom: Always fill out the form starting from the top. If you do it from the bottom, it might mess up the results.

  • Reset Before New Data: Always hit the reset button before you upload new files. This makes sure everything is accurate and there's no data mix-up.

  • Enter Days for Interest: Type in the number of days after which you want to start calculating the interest.

  • Enter Interest Percentage: Enter the percentage rate for the interest.

  • End Date and Files: Pick the date (in dd-mm-yyyy format) until which you want to calculate the interest. Upload the "Challan_info_excel.xlsx" for challan info and "Payment_info_excel.xlsx" for payment info.

  • Submit to Render Table: Click the submit button to process the data and render the interest calculation table.
